Bem-vindo à API do TARS-RISK

Esta é a API de backend para o sistema TARS-RISK, uma ferramenta para gerenciamento de riscos.

Endpoints Disponíveis:

GET /api/status

Verifica o status da API.

POST /api/auth/register

Registra um novo usuário.

POST /api/auth/login

Autentica um usuário.

POST /api/auth/logout

Encerra a sessão do usuário.

GET /api/risks/

Obtém todos os riscos do usuário atual.

POST /api/risks/

Cria um novo risco.

GET /api/risks/{id}

Obtém um risco específico.

PUT /api/risks/{id}

Atualiza um risco existente.

DELETE /api/risks/{id}

Exclui um risco.
